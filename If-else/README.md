
# 🔹 If-Else Case Study Programs

This folder contains Java programs based on **if-else conditional statements**.  
All programs are written as case study questions to help beginners understand decision-making concepts in Java.

---

## 📚 Programs Included

- Leap Year Checker
- Grade Calculator
- Average Marks Program
- Even or Odd Number
- Positive or Negative Number
- Largest Number Finder
- Pass or Fail System
- Percentage Classification

---

## 🎯 Objective

The purpose of these programs is to:

- Learn Java conditional statements
- Improve logical thinking
- Practice real-world case study problems
- Prepare for exams and coding practice

---

## 🛠️ Concepts Used

- if statement
- if-else statement
- else-if ladder
- nested if
- logical operators (&&, ||)

---

## ▶️ How to Run

Compile the program:

javac FileName.java

Run the program:

java FileName

---

## 💻 Language

Java (Console-Based Programs)

---

## 📌 Note

These programs are beginner-friendly and created for practice, assignments, and learning Java fundamentals.
