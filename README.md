
# ☕ Java Case Study Questions

This repository contains a collection of Java case study programs created for learning and practicing core Java programming concepts. The programs are beginner-friendly and help improve logical thinking and problem-solving skills.

---

## 📚 Topics Covered

- If–Else Statements
- Switch Case
- Loops (for, while, do-while)
- User Input using Scanner
- Mathematical Programs
- Grade & Average Calculations
- Leap Year Program
- Basic Logical Problems
- Case Study Based Questions

---

## 🎯 Purpose

This repository is designed to:

- Help beginners learn Java programming
- Practice college practical questions
- Improve coding logic
- Prepare for exams and interviews

---

## 🛠️ Technologies Used

- Java
- JDK (Java Development Kit)
- Console-Based Programs

---

## ▶️ How to Run Programs

1. Install Java (JDK)
2. Clone this repository

git clone https://github.com/your-username/java-case-study.git

3. Compile the program

javac FileName.java

4. Run the program

java FileName
